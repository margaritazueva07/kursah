Данная программа представляет собой консольное приложение на языке C для анализа кусочно-заданной математической функции с интерфейсом на русском языке, предлагающее пользователю 5 основных операций через меню.
Основные функции программы:
main() - Основная функция, реализующая интерактивное меню для выбора операций с обработкой пользовательского ввода.
f(double x) - Вычисляет значение кусочно-заданной функции в точке x, учитывая три различных аналитических выражения для разных интервалов.
derivative(double x, double h, double result)* - Вычисляет производную функции в точке x численным методом центральной разности с заданным шагом h.
find_abs_min(double a, double b, double step, double min_val)* - Находит наименьшее значение функции на отрезке [a, b] путём перебора с заданным шагом.
find_abs_max(double a, double b, double step, double max_val)* - Находит наибольшее значение по модулю функции на отрезке [a, b] путём перебора с заданным шагом.
find_x(double Y, double a, double b, double epsilon) - Решает уравнение f(x)=Y на интервале [a,b] методом бисекции с заданной точностью epsilon.
save_table_to_file(double a, double b, double step, const char filename)* - Сохраняет таблицу значений функции на интервале [a,b] с шагом step в указанный файл.
Созданный инструмент обеспечивает проведение многопланового исследования функциональных зависимостей, интегрируя в себе надежные численные методы и современные алгоритмы обработки данных.
